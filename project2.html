Rocket Jump!, A Simple 2D Platformer

March 2015 - May 2015

The code for this project can be found here.

Rocket Jump! is a very simplistic 2D platformer in which you traverse a war-torn battlefield as a cyborg with a rocket launcher for an arm. With this weapon, you can defeat enemies that stand in your way, or even use the rocket's explosion to reach incredible heights by shooting at the ground as you jump.

Cool Features

One feature that was added to the game was customizeable levels, through an outside level editor. This level editor would generate a .txt file that the game would read in, so you can create and make your own levels, and easily implement it into the game by simply placing the files into the correct directory!

My Role

During the process of making this mini-game, I had worked mainly on implementing the user interface of the game, which included all of the menu screens and the HUD (Heads Up display) of the game. I had also worked on the level editor and how files are read into the game for easy access to the user. Other aspects I had worked on are the artwork of the game, which included the playavble character, enemies, and textures for background and tiles throughout the levels.
